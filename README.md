Blood Pressure Tracker
=======================

## Prerequisites
* Node.js 8.x+
* PostgreSQL

## Installing
* ``` git clone git@github.com:kistasi/blood-pressure-tracker.git ```
* ``` cd  blood-pressure-tracker ``` 
* ``` npm install ```
* Create a PostgreSQL database.
* Update ``` config/config.json ``` with your database credentials.
* ``` npm run start-dev ```
